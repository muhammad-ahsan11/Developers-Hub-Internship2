# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting house prices using machine learning techniques. It analyzes various property features such as area, number of bedrooms, bathrooms, parking, and furnishing status to estimate the final house price.

The model helps understand how different factors influence real estate pricing and demonstrates the use of regression algorithms for real-world prediction tasks.

---

## 🎯 Objective
- To build a regression model for house price prediction  
- To analyze relationships between property features and price  
- To compare different machine learning models  
- To evaluate performance using error metrics  

---

## 📊 Dataset
The dataset includes the following features:
- Area (square footage)
- Bedrooms
- Bathrooms
- Parking spaces
- Furnishing status
- Other house-related attributes  

**Target variable:** Price

---

## 🛠️ Technologies Used
- Python 🐍  
- Pandas & NumPy (Data Handling)  
- Matplotlib (Visualization)  
- Scikit-learn (Machine Learning Models)  

---

## 🤖 Machine Learning Models
- Linear Regression  
- Gradient Boosting Regressor  

---

## 🔄 Workflow
1. Load dataset  
2. Data preprocessing and cleaning  
3. Convert categorical variables into numeric form  
4. Feature scaling  
5. Train-test split  
6. Train models  
7. Make predictions  
8. Evaluate performance (MAE & RMSE)  
9. Visualize results  

---

## 📈 Results
- Gradient Boosting performed better than Linear Regression  
- Lower MAE and RMSE indicate higher accuracy  
- Model effectively captures relationships between features and price  

---

## 📉 Visualizations
- Actual vs Predicted Price graph  
- Error Distribution histogram  
- Feature importance plot  

---

## 🧾 Conclusion
This project demonstrates how machine learning can be used to predict house prices with good accuracy. Gradient Boosting Regressor provided the best performance, showing that advanced ensemble methods work better for this dataset.
